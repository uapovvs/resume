# Launch UI Components

You have access to the Launch UI repository cloned at:
`.agents/skills/launch-ui/repo`

## Usage
When the user requests to use Launch UI components or patterns:
1. Browse the `.agents/skills/launch-ui/repo/components` directory to find requested components (e.g., hero sections, navbars, buttons).
2. Copy the relevant component code into the current project's `components/ui` or equivalent directory.
3. Ensure any required dependencies (like framer-motion, lucide-react, or radix-ui primitives) are installed in the current project before using the components.
4. Update the component's imports to match the current project's structure (e.g., alias `@/components/ui/...`).

You can explore the cloned repository to find inspiration or exact implementations of Launch UI elements.
